Descuento Bancario

Teoría y aplicación de las operaciones de descuento, una herramienta clave para anticipar liquidez.

Fundamentos del Descuento

El descuento es una operación financiera que consiste en **anticipar el valor de un título-valor** (como una letra de cambio o un pagaré) que tiene un vencimiento futuro. A cambio de adelantar el dinero, la entidad financiera cobra un interés por adelantado, el cual se denomina "descuento".

Variables Clave

  • Valor Nominal (S): Es el valor escrito en el documento, el monto que se pagaría en la fecha de vencimiento.
  • Valor Actual o Líquido (P): Es el dinero que efectivamente se recibe hoy después de restar el descuento.
  • Descuento (D): Es el interés cobrado por adelantado por la entidad financiera. Se cumple que D = S - P.
  • Tasa de Descuento (d): Es la tasa de interés que se aplica para calcular el descuento.
  • Tiempo (n): Es el plazo que falta para el vencimiento del título, contado desde el día del descuento.

Tipos de Descuento

Existen dos métodos principales para calcular el descuento, y la diferencia entre ellos es fundamental:

  • Descuento Racional (o Matemático): El descuento se calcula sobre el Valor Actual (P). Es el método más justo desde el punto de vista matemático, ya que el interés se calcula sobre el dinero que realmente se presta. Sin embargo, es el menos común en la práctica comercial.
  • Descuento Bancario (o Comercial): El descuento se calcula sobre el Valor Nominal (S) del título. Es el más utilizado por las entidades financieras porque genera un mayor rendimiento para ellas.
